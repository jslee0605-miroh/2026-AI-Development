# Junior Software Engineer - Python

**Location:** Hybrid (Chicago, IL or Remote)
**Department:** Engineering
**Employment Type:** Full-Time
**Experience Level:** Entry Level (0-2 years)

## About the Role

We are looking for a motivated Junior Software Engineer to join our growing team. This role is perfect for recent graduates, bootcamp alumni, or early-career developers who are passionate about building web applications with Python. You'll work alongside experienced engineers, contribute to real projects from day one, and have opportunities to learn and grow your technical skills.

## What You'll Do

- Write clean, maintainable Python code for backend services and APIs
- Contribute to the development of web applications using Django or Flask
- Work with databases to store and retrieve application data
- Collaborate with team members through code reviews and pair programming
- Participate in Agile ceremonies (standups, sprint planning, retrospectives)
- Write unit tests and help maintain code quality
- Learn from senior engineers and contribute to team knowledge sharing
- Debug issues and implement fixes under guidance

## Required Qualifications

### Technical Skills
- **Python Fundamentals:**
  - Solid understanding of Python 3.x
  - Experience with Python data structures (lists, dicts, sets, tuples)
  - Understanding of OOP concepts (classes, inheritance, encapsulation)
  - Familiarity with Python standard library

- **Web Development:**
  - Basic understanding of HTTP, REST APIs, and JSON
  - Exposure to at least one Python web framework (Django, Flask, FastAPI)
  - HTML/CSS fundamentals
  - Basic JavaScript knowledge is a plus

- **Databases:**
  - Basic SQL knowledge (SELECT, INSERT, UPDATE, DELETE, JOIN)
  - Understanding of relational database concepts
  - Experience with PostgreSQL, MySQL, or SQLite

- **Version Control:**
  - Familiarity with Git (clone, commit, push, pull, branch)
  - Understanding of basic Git workflows

- **Development Tools:**
  - Comfortable with command line/terminal
  - Experience with a code editor or IDE (VS Code, PyCharm, etc.)

### Professional Skills
- Bachelor's degree in Computer Science, related field, or equivalent practical experience
- OR completion of coding bootcamp or self-taught with portfolio of projects
- 0-2 years of professional software development experience (internships count!)
- Strong problem-solving skills and attention to detail
- Good communication skills and ability to work in a team
- Eagerness to learn and grow as a developer
- Ability to receive and act on constructive feedback

## Nice to Have (But Not Required!)

- **Additional Python Skills:**
  - Experience with virtual environments (venv, virtualenv)
  - Familiarity with package management (pip, poetry)
  - Knowledge of Python testing frameworks (pytest, unittest)
  - Understanding of async/await and asynchronous programming

- **Web & API Development:**
  - Built a REST API using Django REST Framework or Flask
  - Understanding of authentication (JWT, sessions)
  - Exposure to frontend frameworks (React, Vue, Angular)

- **Data & Databases:**
  - Experience with ORMs (Django ORM, SQLAlchemy)
  - Basic understanding of database design and normalization
  - Exposure to NoSQL databases (MongoDB, Redis)

- **DevOps & Tools:**
  - Familiarity with Docker basics
  - Exposure to cloud platforms (AWS, Google Cloud, Azure)
  - Experience with CI/CD pipelines
  - Linux/Unix command line experience

- **Other:**
  - Personal projects or open source contributions on GitHub
  - Technical blog posts or documentation writing
  - Participation in hackathons or coding competitions

## What Success Looks Like

**First 30 Days:**
- Complete onboarding and set up development environment
- Make your first code contribution (small bug fix or feature)
- Understand our codebase structure and development workflow
- Meet the team and attend all sprint ceremonies

**First 90 Days:**
- Independently complete assigned tickets with code reviews
- Write tests for your code
- Contribute to team discussions and code reviews

**First Year:**
- Own small features end-to-end
- Mentor new team members during onboarding
- Participate in technical decision-making
- Demonstrate growth in technical skills and autonomy

## Interview Process

1. **Phone Screen** (30 minutes): Chat with recruiting team about your background and interests
2. **Technical Screen** (60 minutes): Solve 1-2 coding problems in Python (think LeetCode Easy/Medium)
3. **On-site/Virtual Interview** (3 hours):
   - Coding exercise: Build a small feature or fix bugs in a sample project
   - Technical discussion: Review your past projects or code samples
   - Behavioral interview: Team fit and problem-solving approach
   - Q&A: Ask us anything about the role, team, or company
4. **Final Chat** (30 minutes): Brief conversation with hiring manager
